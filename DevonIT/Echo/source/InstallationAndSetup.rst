========================================
Virtual Appliance Installation and Setup
========================================

This administration guide outlines how to install, setup, and run the
Echo™ Management Platform. Required components include:

#. A 64-bit host running VMware® Workstation Player™, VMware® ESXi™, Citrix® 
   XenServer®, or Microsoft® Hyper-V®.
#. Access to DNS and DHCP server configuration.

The instructions below are for users who wish to use an alternate method to 
the OVF deployment that is available.
