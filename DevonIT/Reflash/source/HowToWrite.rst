Writing Disk Images to Thin Clients
===================================

This guide describes how to create a re-imaged utility on a USB Flash drive and 
use it to write a disk image to Lenovo thin clients. 

The following hardware is required:

+  A Windows or Linux system with access to the Internet
+  A USB Flash drive with at least 4 GB of storage
+  A Lenovo thin client
